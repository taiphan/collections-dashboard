# Changelog — Collections Dashboard

## v1.0.0 (2026-05-31)

### Features
- Full collections case management dashboard
- Real-time analytics with interactive charts (Recharts)
- Agent performance tracking and scoring
- Compliance monitoring module
- Strategy configuration and management
- Multi-theme support (light/dark/system)
- Responsive layout with shadcn/ui components
- Zustand state management with persistence
- CSV data import/export via PapaParse

### UI/UX
- Professional SVG favicon (green dollar/collection icon)
- Standardized design system with Base UI + CVA
- Accessible components with ARIA labels
- Smooth transitions and hover states

### Infrastructure
- Next.js 16 App Router with Turbopack
- TypeScript strict mode
- Tailwind CSS v4
- Port: 3010 (frontend)
